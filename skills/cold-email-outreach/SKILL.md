# Cold Email Outreach Expert — Skill Reference

Quick reference for cold email outreach frameworks. These files distill the targeting strategies, infrastructure setup, copy templates, Instantly integration patterns, and optimization rubrics used by the Cold Email Outreach Expert agent.

## When to Use

Use these frameworks when:
- Building a cold email campaign from scratch (targeting, infrastructure, copy)
- Optimizing existing cold email campaigns for better deliverability and reply rates
- Setting up email infrastructure (domains, warmup, authentication)
- Writing cold email sequences using proven copy frameworks
- Managing campaigns via the Instantly MCP integration
- Auditing campaign performance against benchmarks

## Skill Files

| File | What It Covers |
|------|----------------|
| `targeting-and-lists.md` | ICP definition, list building strategies, lead sourcing, segmentation, list hygiene |
| `email-infrastructure.md` | Domain setup, SPF/DKIM/DMARC, warmup protocols, sending limits, deliverability |
| `copy-frameworks.md` | Email sequence templates, subject lines, CTAs, personalization, A/B testing |
| `instantly-playbook.md` | Instantly MCP integration, campaign creation, lead management, analytics |
| `campaign-optimization.md` | Performance benchmarks, reply rate optimization, deliverability troubleshooting |
