# Landing Page Review Checklist

Section-by-section scoring rubric, completeness check, copy quality gate, and common mistakes.

Source: `make-high-converting-landing-pages/02-through-14.md`, `copywriting-fundamentals/02-rules.md`, `landing-page-ab-tests-getting-started/04-gather-data.md`

---

## Section-by-Section Scoring Rubric

Score each section Red / Yellow / Green:

- **Green**: Strong — clear, compelling, well-structured, no major issues
- **Yellow**: Functional but has improvement opportunities
- **Red**: Missing, broken, or actively hurting conversion

---

### 1. Navbar

| Criteria | Green | Yellow | Red |
|----------|-------|--------|-----|
| Sticky | Stays visible on scroll | Present but not sticky | Missing |
| CTA in nav | CTA button visible in navbar | Link but no button | No CTA in nav |
| Simplicity | Essential links only | Some clutter | Overwhelming navigation |

### 2. Hero

| Criteria | Green | Yellow | Red |
|----------|-------|--------|-----|
| Header clarity | Reader knows exactly what you sell from header alone | Mostly clear but requires subheader | Vague, slogany, or confusing |
| Header specificity | Includes specific claim or objection-handler | Descriptive but generic | Could apply to any competitor |
| Subheader | Complements header — explains how/why | Redundant with header | Missing or too long |
| CTA | Clear, action-oriented, continues narrative | Present but generic ("Submit") | Missing or confusing |
| Visual | Shows product in action or represents value | Stock photo, somewhat relevant | Missing, irrelevant, or generic illustration |
| Hook type | Uses 2-3 hook types effectively | Uses 1 hook type | No hook — just a statement |

### 3. Social Proof #1 (Logos)

| Criteria | Green | Yellow | Red |
|----------|-------|--------|-----|
| Present | Logos of recognizable customers/press | Present but unknown logos | Missing entirely |
| Placement | Above fold or immediately after hero | Below fold | Buried at bottom |
| Quantity | 4-8 logos or compelling stat | 1-3 logos | None |

### 4. Features / Benefits / Objections

| Criteria | Green | Yellow | Red |
|----------|-------|--------|-----|
| Benefit-focused | Headers state customer value, not feature names | Mix of benefits and features | All feature labels, no benefits |
| Objection handling | Each block addresses one objection | Some objections addressed | No objection handling |
| Visual support | Each block has relevant image/GIF | Some blocks have images | Text only |
| CTA per block | Tailored CTA on each feature block | One generic CTA at end | No CTAs in section |
| Narrative | Ties back to hero value prop | Loosely connected | Disconnected from hero |
| Count | 3-6 focused blocks | 1-2 or 7+ | Missing entirely |

### 5. Social Proof #2 (Testimonials)

| Criteria | Green | Yellow | Red |
|----------|-------|--------|-----|
| Specificity | Testimonials highlight specific value received | Generic praise ("Great product!") | Missing |
| Trust signals | Photo + name + company/title | Name only | Anonymous |
| Objection handling | At least one testimonial addresses a common objection | None address objections | N/A |
| Relevance | Testimonials match target persona | Somewhat relevant | Clearly different audience |

### 6. FAQ

| Criteria | Green | Yellow | Red |
|----------|-------|--------|-----|
| Completeness | Covers objections, pricing, who it's for/not for, support | Covers some but not all | Missing or just 1-2 items |
| Benefits repeated | Key benefits restated in FAQ format | Some overlap | No benefit reinforcement |
| Objection coverage | All major objections addressed | Some gaps | Major objections unanswered |

### 7. CTA Section

| Criteria | Green | Yellow | Red |
|----------|-------|--------|-----|
| Value reiteration | Restates strongest value prop | Generic CTA only | Missing section |
| Action clarity | Clear what happens next (start trial, buy, etc.) | Somewhat clear | Vague or confusing |
| Urgency / reason | Includes reason to act now | No urgency | N/A |
| Traffic match | CTA matches traffic temperature (cold=trial, warm=buy) | One-size-fits-all | Mismatched (cold traffic → buy now) |

### 8. Footer

| Criteria | Green | Yellow | Red |
|----------|-------|--------|-----|
| Legal | Privacy policy + terms linked | Partially present | Missing |
| Key pages | Links to pricing, about, contact | Some links | No useful links |
| Distraction | Minimal for conversion-focused pages | Some unnecessary links | Full site nav competing with CTA |

## 7-Question Completeness Check

Walk through the page and verify each question is answered:

| # | Question | Where It Should Be Answered |
|---|----------|---------------------------|
| 1 | What is the product? | Hero header + subheader |
| 2 | Is it right for me? | Hero + feature blocks (persona signals) |
| 3 | Is it legit? | Design quality + social proof #1 |
| 4 | Who else is using it? | Social proof #1 + #2 |
| 5 | How much? | Pricing section or FAQ |
| 6 | Where can I get help? | FAQ + footer (contact link) |
| 7 | How do I take action? | CTA sections (hero + dedicated + feature blocks) |

Score: How many of the 7 are clearly answered?
- 7/7 = Green
- 5-6/7 = Yellow
- <5/7 = Red

## Copy Quality Gate (Harry Dry's 3 Rules)

Apply to the hero header, each feature header, and the CTA:

| Rule | Test | Pass | Fail |
|------|------|------|------|
| **Visualize** | Can you picture it concretely? | "1,000 songs in your pocket" | "32GB storage capacity" |
| **Falsify** | Can the claim be proven true/false? | "Reads every day" | "Quality you can trust" |
| **Uniqueness** | Can nobody else say this? | Unique to your product/story | Generic — any competitor could claim it |

**Three fails = rewrite the copy before anything else.**

## 5-Category Self-Assessment

Before any changes, audit the page across these categories (from A/B testing curriculum):

### Relevancy
Does the page deliver the experience the visitor expected based on how they arrived? Ad copy matches landing page copy? Search query matches page content?

### Clarity
Is the content immediately understandable? Any jargon? Any sections requiring re-reading? Any confusion about what the product does?

### Value
Does the page present product value compellingly? Room for improvement in how benefits are communicated? Are value props specific enough?

### Friction
What parts cause confusion, doubt, or hesitation? Complex forms? Unclear pricing? Missing information? Too many choices?

### Distraction
What elements distract from the main CTA? Competing links? Unnecessary content? Visual clutter? Auto-playing videos?

## Common Mistakes (Instant Red Flags)

1. **Slogany header** — "Empower your workflow!" tells nobody anything
2. **Feature-first copy** — Listing what the product does instead of what the customer gets
3. **No social proof** — Instant trust deficit
4. **Generic stock photos** — Random astronaut illustrations instead of product screenshots
5. **Hidden CTA** — Must scroll to find the action button
6. **Walls of text** — No visual hierarchy, no bold, no lists
7. **Missing objection handling** — Leaving the biggest "but what about..." unanswered
8. **No mobile optimization** — Broken layout or unreadable text on phones
9. **CTA/traffic mismatch** — Asking cold traffic to "Buy now"
10. **No pricing information** — Wasting time on leads who can't afford it
