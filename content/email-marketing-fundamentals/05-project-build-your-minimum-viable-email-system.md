# Project: Build your minimum viable email system

By the end of this module, you should have:

## ESP setup completed

- [ ]  Account created in MailerLite or Mailchimp
- [ ]  Domain verified (may take 24-48 hours)
- [ ]  Confirm **SPF, DKIM, DMARC** records are set up
- [ ]  Sender details configured (name, email, address)
- [ ]  Test email sent and received successfully
- [ ]  Main subscriber list created

## Essential email flows in place

#### **Transactional emails** 

Sent from a separate subdomain/domain:

- [ ]  Signup confirmation email working and branded
- [ ]  Password reset email working and branded
- [ ]  Any other automated emails your app sends

#### **Onboarding sequence**

- [ ]  Basic welcome email created (or completed in Onboarding module)
- [ ]  Sequence triggers set up in your ESP
- [ ]  Test emails sent to yourself

#### **Win-back flow (only if 100+ users/month)**

- [ ]  2-3 email sequence written using provided templates
- [ ]  Automation triggers set up (7-day, 14-day inactive)
- [ ]  Test sequence activated

## Content created with templates

- [ ]  Subject lines written using Story System hooks
- [ ]  Email copy follows proven structure (Hook, Explanation, Proof, CTA)
- [ ]  CTAs match your standard language from Story System
- [ ]  Brand voice consistent across all emails

## Tracking and measurement

- [ ]  UTM parameters set up for email links
- [ ]  Monthly review process scheduled in calendar
- [ ]  Baseline metrics recorded (deliverability, open rate, click rate)
- [ ]  List cleaning process understood

## Quick troubleshooting guide

#### "My domain verification isn't working"

- Wait 24-48 hours after adding DNS records
- Double-check you added the records to the right domain
- Contact your domain provider's support if still stuck

#### "My emails are going to spam"

- Make sure you're using a verified domain
- Avoid spam trigger words in subject lines
- Send from a person's name, not "noreply"
- Keep your unsubscribe rate under 2%

#### "My open rates are terrible"

- Test different subject lines
- Check your sender name (use a real person)
- Clean your email list of inactive subscribers
- Send at different times

#### "I can't set up automation triggers"

- Look for "Automations" or "Workflows" in your ESP
- Start with pre-built templates if available
- Search "[your ESP] automation tutorial" on YouTube
