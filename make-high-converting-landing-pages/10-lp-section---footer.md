# LP Section – Footer

**If needed, to put this back into context of the page, see "Footer" below.**

![](https://cdn.prod.website-files.com/5ca4f46f1600f04b73b1ccfe/683506aa08e3200b82786621_1744225409912.avif)

You can generally keep it pretty simple:

- Copyright info
- Links to privacy policy and terms (legally required)
- Links to key pages (pricing, dedicated feature pages, about, contact, etc)
- If this landing page is your homepage, you’ll probably want to dump all the links for your site here like for blog, newsletter, etc.
- If it’s for a landing page for an ad or dedicated funnel, strip out all the non-essential links.

> **If this is a conversion-focused landing page, now is not the time to link to 100 different pages. Don't let visitors get lost in the maze.**
